# Multi_Diagnostic
Stochastic Feature Integration: A Computationally Efficient Probabilistic Framework for Multi-Diagnostic Thoracic Imaging
Stochastic Feature Integration (SFI) for Thoracic Imaging
This repository contains the official code for the paper: "Stochastic Feature Integration: A Computationally Efficient Probabilistic Framework for Multi-Diagnostic Thoracic Imaging".
How to run:

Install requirements: pip install -r requirements.txt

Update the base_dir variable in main.py to point to your image dataset.

Run the script: python main.py
